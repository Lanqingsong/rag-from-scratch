## 知识库文档的挑战

把文件内容读成"干净的文本"比想象中难，不同格式有各自的噪声：

| 格式 | 常见问题 |
|------|---------|
| PDF | 换行断词、页码/页眉污染、双栏布局乱序、图片无文字 |
| DOCX | 样式标签、空白段落、页眉页脚混入 |
| TXT | 编码问题（GBK vs UTF-8） |
| MD | Markdown 语法残留（`**` `$$` 等） |

---

## PDF 解析

### 方式一：pdfminer（本示例使用）

```python
from pdfminer.high_level import extract_pages
from pdfminer.layout import LTTextContainer

def extract_text_from_pdf(filename, page_numbers=None, min_line_length=1):
    paragraphs = []
    buffer = ""
    full_text = ""

    for i, page_layout in enumerate(extract_pages(filename)):
        if page_numbers is not None and i not in page_numbers:
            continue                            # 只提取指定页
        for element in page_layout:
            if isinstance(element, LTTextContainer):  # 只要文本块
                full_text += element.get_text() + "\n"

    # 重组段落：短行视为段落分隔符
    for line in full_text.split("\n"):
        if len(line) >= min_line_length:
            buffer += " " + line if not line.endswith("-") else line.strip("-")
        elif buffer:
            paragraphs.append(buffer.strip())
            buffer = ""

    if buffer:
        paragraphs.append(buffer.strip())
    return paragraphs
```

**连字符处理：**  
英文 PDF 中长词常被截断为 `know-\nledge`，代码把行尾 `-` 识别为连字符，拼接时去掉它。

### 方式二：LangChain PyPDFLoader（主项目使用）

```python
from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("manual.pdf")
docs = loader.load()   # 每页一个 Document 对象

for doc in docs:
    print(doc.metadata)      # {'source': 'manual.pdf', 'page': 0}
    print(doc.page_content)  # 该页的文本内容
```

**优势：** 自动附带 `source` 和 `page` 元数据，方便检索后溯源。

## DOCX 解析

```python
from docx import Document

def extract_text_from_docx(filename, min_line_length=1):
    paragraphs = []
    buffer = ""

    doc = Document(filename)
    for para in doc.paragraphs:
        line = para.text
        if len(line) >= min_line_length:
            buffer += " " + line
        elif buffer:
            paragraphs.append(buffer.strip())
            buffer = ""

    if buffer:
        paragraphs.append(buffer.strip())
    return paragraphs
```

**注意：** `doc.paragraphs` 只提取正文，页眉页脚需要单独处理（`doc.sections`）。

## 本项目的文档清洗

`splitters.py` 中的 `_clean()` 函数在切分前对文本做预处理：

```python
def _clean(text: str) -> str:
    # 去除作者水印
    text = re.sub(r'by\s+\d+@qq\.com\s*持续更新中…?', '', text)
    # 去除 Markdown 加粗符号（但保留内容）
    text = re.sub(r'\*\*(.*?)\*\*', r'\1', text)
    # 去除 Mermaid 流程图代码块（无法向量化）
    text = re.sub(r'```mermaid[\s\S]*?```', '', text)
    # 合并多余空行
    text = re.sub(r'\n{3,}', '\n\n', text)
    return text.strip()
```

## 元数据的重要性

LangChain 的 `Document` 对象包含 `page_content` 和 `metadata` 两个字段。  
`metadata` 中记录来源信息，检索后可以展示给用户"这条答案来自哪个文件第几页"：

```python
Document(
    page_content="焦距 f = WD × sensor_size / FOV",
    metadata={
        "source": "knowledge_base/机器视觉第一部分.md",
        "section_title": "2.1 镜头选型计算"
    }
)
```

本项目在 `app.py` 的 `extract_references()` 函数中提取这些元数据，  
渲染在前端的"参考资料"折叠面板中。
